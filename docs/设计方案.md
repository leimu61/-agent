# 如意学伴（Ruyi Study Buddy）融合版项目设计方案

> 北京科技大学 2026 年生产实习 · 人工智能方向  
> 课题：Hermes Agent 智能学习规划助手  
> 组员：4 人 | 周期：4 天 | 技术栈：Hermes Agent（Agent 驱动开发 + SKILL.md + 轻量工具层）

## 一、项目概述

### 1.1 一句话描述

基于 Hermes Agent 的三角色智能学习管家，以 SKILL.md 为主骨架，配合艾宾浩斯等轻量认知工具（由 Agent 生成），制定自适应学习计划、追踪进度、个性化答疑，支持微信定时推送和离线导出。

### 1.2 核心亮点

- 🤖 **Agent 驱动开发**：全部代码由 Hermes 生成，组员零手写
- 🎭 **三角色系统**：规划师、答疑导师、督学伙伴三种模式，按场景自动切换
- ⚙️ **认知工具增强**：艾宾浩斯遗忘曲线、计划约束校验等轻量 Python 工具由 Agent 自动生成
- 📱 **微信提醒 + 终端快速响应**：定时推送提醒，回终端说一句话更新进度
- 📄 **离线导出**：周报 PDF/Markdown，学习数据 JSON
- 🔒 **隐私安全**：数据 100% 本地存储，不上传任何第三方
- 💾 **个性化 Memory**：学习风格、薄弱知识图谱跨会话持久化
- ⏱ **4 天可交付**：从零到完整演示

**全链路实现说明：**

| 组件 | 作用 | 实现方式 | 谁写代码 |
|------|------|----------|:--:|
| SKILL.md | 三角色行为定义 + 约束规则 | Markdown 文本 | Hermes |

### 2.1 工具层设计（由 Hermes Agent 生成）

三个轻量 Python 工具，总计约 250 行。**组员提供规则描述，Hermes 自动生成代码，组员验证跑通即可。**

**① 艾宾浩斯日程计算器** (`tools/ebbinghaus.py`)
- 输入：知识点首次学习日期
- 输出：基于 1, 2, 4, 7, 15 天间隔的复习日期列表
- 注册为 Hermes Tool，规划器调用后自动在计划中插入复习任务

**② 计划约束校验器** (`tools/constraint_checker.py`)
- 输入：待生成的任务列表
- 校验规则：日总时长不超限 / 单科连续不超 2h / 超限自动缩放
- 输出：校验通过的计划或修正建议

**③ 错题检索器** (`tools/mistake_retriever.py`)
- 输入：当前知识点关键词
- 在 Memory 中匹配历史错题
- 返回相关错题及错误次数，供答疑导师附加提醒

**更新规则：**

| 事件 | 触发条件 | 更新操作 |
|------|----------|----------|
| 生成新计划 | 用户说"制定计划" | 覆盖 `plan.tasks`，生成 `ebbinghaus_queue` |
| 汇报完成 | 用户说"完成了XX" | 更新 task 状态/评分 |
| 发现薄弱点 | 答疑时说"没懂" | `weak_points` 累加 |
| 生成周报 | 用户说"本周总结" | 计算 `weekly_stats` |
| 修改推送 | 用户说"早报改9点" | 更新 `push_schedule` |

### 2.3 微信提醒 + 终端快速响应

```
Cron 定时 → Hermes 生成内容 → Gateway → 企业微信 Webhook → 手机微信收到
学生看微信 → 回终端说"已完成数据结构" → Agent 更新 Memory → 回复确认
```

推送文案示例：

```
📚 如意学伴 · 今日早报
━━━━━━━━━━━━━
📅 2026-07-20 周一
🔹 数据结构：线性表（2h）
🔄 艾宾浩斯复习：链表（0.5h）
━━━━━━━━━━━━━
💬 完成某项后，回终端说"已完成数据结构"更新进度
```

### 2.4 自定义推送时段

学生可随时调整："早报改到9点" → 更新 `push_schedule`。

### 2.5 离线导出

```bash
hermes chat -q "生成周报保存到桌面/周报.md"
hermes chat -q "生成PDF周报到桌面"
hermes chat -q "导出全部学习数据到桌面/学习数据.json"
```

## 八、答辩演示脚本（3 分钟）

**开场（30s）**
> "如意学伴"——Agent 驱动开发的三角色 AI 学习管家。我们不写代码，我们指挥 AI 写代码。

**第一幕（1min）**：规划师出计划 + 艾宾浩斯自动插入复习 + 答疑导师费曼讲解 + 错题提醒。

**第二幕（1min）**：学生说"已完成数据结构"→ 督学秒回确认。掏出手机展示微信早晚推送。

**第三幕（30s）**：桌面展示 PDF 周报（含复习日历）。总结：Agent 驱动开发、三角色、认知工具、4天交付。
